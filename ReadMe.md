"# Flipkart Automation Framework" 
## Overview
Automates Flipkart buy-product flow:
- Login with mobile and manual OTP
- Search for a dynamic product
- Add product to cart
- Enter delivery pincode

---

## Tech Stack
- Java + Selenium WebDriver
- TestNG + Maven
- Page Object Model (POM) design pattern
- Data-driven using TestNG DataProvider
- Screen recording with OBS Studio / Monte Recorder
